# hr_analytics_sql
 This repository contains an SQL-based analysis   "hr_analytics" dataset. The objective of this project is to analyze employee data to understand patterns related to employee attrition, salary trends, department distribution, and employee satisfaction.  In this project, multiple SQL queries are used to extract meaningful insights from the dataset.
The analysis includes metrics such as total employees, attrition rate, department-wise employee distribution, salary analysis, job role distribution, and the impact of overtime on employee attrition.

The project demonstrates the use of important SQL concepts such as GROUP BY, CASE WHEN, aggregate functions (COUNT, AVG, MAX), subqueries, and filtering to perform data analysis on an HR dataset.

This project is useful for beginners and aspiring data analysts who want to practice real-world HR data analysis using SQL and build a strong portfolio for learning and career growth.
