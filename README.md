# hr_analytics_sql

## Project Overview

This project focuses on analyzing HR employee data using **SQL queries** with an **Excel dataset** as the data source. The main objective of this project is to explore employee data and identify important patterns related to **employee attrition, salary distribution, department structure, job roles, and employee satisfaction**.

The project demonstrates how raw HR data can be transformed into meaningful insights by combining **Excel for data storage** and **SQL for data analysis**.

---

# Data Source

The dataset used in this project is an **HR employee dataset stored in Excel format**. This dataset contains different attributes that describe employee details and workplace conditions.

Important columns included in the dataset:

* EmpID – Unique identifier for each employee
* Age and AgeGroup – Employee age information
* Department – Department where the employee works
* JobRole – Employee job position
* MonthlyIncome – Employee monthly salary
* Attrition – Indicates whether an employee left the company or not
* JobSatisfaction – Employee satisfaction with their job
* EnvironmentSatisfaction – Satisfaction with the work environment
* OverTime – Whether the employee works overtime
* DistanceFromHome – Distance between employee home and workplace
* NumCompaniesWorked – Number of companies the employee worked for previously

---

# Project Workflow

### 1. Data Collection

HR employee data is collected and stored in an **Excel file**.

### 2. Data Import

The Excel dataset is imported into a **SQL database** and stored in a table named **hr_analytics**.

### 3. Data Exploration

Basic SQL queries are used to explore the dataset and understand its structure.

### 4. Data Analysis

Advanced SQL queries are written to analyze employee trends, salary patterns, department distribution, and attrition behavior.

### 5. Insight Generation

The results obtained from SQL queries are used to generate meaningful HR insights.

---

# SQL Concepts Used

This project demonstrates several important SQL concepts used in real-world data analysis:

* SELECT statements to retrieve data
* WHERE clauses for filtering data
* GROUP BY to summarize data by department or job role
* Aggregate functions such as **COUNT(), AVG(), MAX(), SUM()**
* CASE statements for conditional calculations
* Subqueries for advanced analysis
* ORDER BY to sort and identify top results

---

# Analysis Performed

The following HR-related analyses were performed using SQL queries:

* Total number of employees in the company
* Attrition analysis (employees who left vs employees who stayed)
* Department-wise employee distribution
* Department-wise attrition trends
* Average salary across departments
* Job role distribution in the organization
* Identification of top highest-paid employees
* Impact of overtime on employee attrition
* Job satisfaction and environment satisfaction analysis
